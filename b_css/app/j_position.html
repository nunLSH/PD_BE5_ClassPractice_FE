fixed : top

position : element 가 배치되는 위치를 지정하는 속성

  static : 기본값
    부모 요소의 좌측 상단 부터 순차적으로 배치
    top, left, right, bottom, z-index 속성 사용 불가

  relative
    부모 요소의 좌측 상단 부터 순차적으로 배치
    top, left, right, bottom, z-index 속성 사용 가능
    요소가 배치된 위치를 기준으로 위치를 지정

  absolute
    부모요소를 기준으로 위치를 지정

  fixed
    화면을 기준으로 위치를 지정
position: relative; left:10vw;
position: relative; right:10vw;
position: relative; top:5vw;
position: relative; bottom:5vw;
position: absolute; left:10vw;
position: absolute; right:10vw;
position: absolute; top:10vw;
position: absolute; bottom:10vw;
