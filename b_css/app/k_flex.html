flex container

    container : 정렬할 요소들을 감싸는 부모요소
      display: flex|inline-flex 지정을 통해 container 로 만들 수 있다.
        display: flex         container 가 block 요소
        display: inline-flex  container 가 inline 요소

      flex-direction : item 이 정렬되는 방향
        row (왼오), row-reverse(오왼), column(위아래), column-reverse(아래위)

    item : container 의 자식요소
